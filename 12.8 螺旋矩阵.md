# 题目：

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

&nbsp;示例 1：

输入：n = 3

输出：\[\[1,2,3],\[8,9,4],\[7,6,5]]



示例 2：

输入：n = 1

输出：\[\[1]]



# 解法：

首先明确，这就是一道模拟的题目，主要考察对于代码的掌控程度。

对于这个题目，我们只需要明确两点：

1\.循环的次数一定是n/2。

2\.每次一个方向的模拟，最后的点都不处理，交给下一个方向的模拟处理，体现在代码上，就是for(;i<n-offset;i++)这里面的n-offset这个位置，就是最后一个点的位置，初始值是n-1.

由于第二点，我们考虑n为奇数的情况，此时需要单独的逻辑来给这个最后的单独剩下的中心点赋值。

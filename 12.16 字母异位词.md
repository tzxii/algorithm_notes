## 题目：

242\. 有效的字母异位词



给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的 字母异位词。



## 解法和代码：

其实一开始就是想着用map的，但是我发现了，题目给定的单词只有小写字母，那么可以直接使用数组来进行简单的哈希。

如果暴力解决，那么就是对于s的每一个单词都在t里面找，找到了就把t里面的那个单词改成“#”



如果用vector来哈希，那么直接统计s和t的单词出现次数就行。

#### 代码：

bool isAnagram(string s, string t) {

&nbsp;       // 由于只包含小写字母，我们完全可以直接使用数组来统计各个字母出现频率

&nbsp;       vector<int> vec1(26,0);

&nbsp;       vector<int> vec2(26,0);

&nbsp;       if(s.length() != t.length()) return false;

&nbsp;       for(int i=0;i<s.length();i++){

&nbsp;           vec1\[s\[i]-'a']++;

&nbsp;           vec2\[t\[i]-'a']++;

&nbsp;       }

&nbsp;       for(int i=0;i<26;i++){

&nbsp;           if(vec1\[i]!=vec2\[i]){

&nbsp;               return false;

&nbsp;           }

&nbsp;       }

&nbsp;       return true;

&nbsp;   }

# 题目

203\. 移除链表元素

给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。

示例 1：

输入：head = \[1,2,6,3,4,5,6], val = 6

输出：\[1,2,3,4,5]

示例 2：

输入：head = \[], val = 1

输出：\[]

示例 3：

输入：head = \[7,7,7,7], val = 7

输出：\[]

提示：

列表中的节点数目在范围 \[0, 104] 内

1 <= Node.val <= 50

0 <= val <= 50

# 解法：

本题就是经典的考虑dummyhead的题目，如果不用dummyhead，那么就要单独判断是否是在删除头节点，其实也就是多一个逻辑判断而已。

但是如果使用dummyhead之后，整个代码的泛化性明显就好多了。此时需要注意的一点是：我们操作的对象是cur->next，因为cur一开始就是dummyhead，而真正有意义的head一开始就是cur->next。

然后我们删除的时候，实际上是更新了cur->next，cur->next = cur->next->next，这不就说明cur->next是被删除了么。

然后如果不删除，那么cur = cur->next，其实是靠更新cur的同时，顺带更新了cur->next.

# 题目：

19\. 删除链表的倒数第 N 个结点

提示

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。



## 代码与解法：

1\.首先注意此时的链表不为空，这是条件的一部分。

2\.删除节点，需要找到其前一个节点。也就是应该找到倒数n+1个节点，我们的fast应该比slow多走n+1步。

3\.此时我们仍然注意：如果删除的是head，那么也需要找到其前一个节点，所以我们仍然设置dummyhead。



ListNode\* removeNthFromEnd(ListNode\* head, int n) {

&nbsp;      // 首先，如果要删除倒数第n个节点，那么其实我们需要找到其前一个节点，也就是倒数n+1的节点

&nbsp;      // 设置dummyhead是为了删除头结点方便，因为删除头节点其实也需要找到头节点的前一个节点

&nbsp;      ListNode\* dummyhead = new ListNode(0);

&nbsp;      dummyhead->next = head;

&nbsp;      ListNode\* slow = dummyhead;

&nbsp;      ListNode\* fast = dummyhead;

&nbsp;      for(int i=0;i<=n;i++){

&nbsp;           fast = fast->next;// 这里我们是去寻找倒数n+1个节点，所以fast应该多走n+1步

&nbsp;      }

&nbsp;      while(fast!=nullptr){

&nbsp;           slow = slow->next;

&nbsp;           fast = fast->next;

&nbsp;      }

&nbsp;       slow->next = slow->next->next;

&nbsp;       return dummyhead->next;

&nbsp;   }
